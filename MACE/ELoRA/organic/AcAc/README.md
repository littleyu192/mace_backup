## 训练

训练仅在 300K 的数据集中进行，测试集包括 300K 600K。

finetune/ 是全参数微调，LoRA/ 是 ELoRA 微调。

编号代表随机数种子。

## 测试

使用训练好的模型在测试集上进行 Energy 和 Force 的计算，然后统计测试集上的误差。

推理和统计相关文件在 [scripts](../../scripts/) 中。

使用 [test.py](./test.py) 计算三次运行的均值和方差。
